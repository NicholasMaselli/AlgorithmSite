NP-Complete

NP-Complete problems are a set of problems that are the hardest problems in the class of NP. NP problems are problems that if given a solution, can be checked in polynomial time but there is no known algorithm to solve these problems in polynomial time. This is the case for a deterministic turing machine (DTM) with a set of states and rules that govern how the machine moves to the next state. Using a theoretic Non-determistic turing machine (NTM) it is possible to solve NP-Complete problems in polynomial time. The NTM can be thought of as "the luckiest decision machine" always chosing the best possible path when given a series possible state transitions. Some NP-Complete problems include:

  • Boolean Satisfiability Problem (SAT)
  • Vertex Cover Problem
  • Hamiltonian Cycle Problem
  • Traveling Salesman Problem
  • Knapsack Problem
