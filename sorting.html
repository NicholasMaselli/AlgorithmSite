Sorting

This page will go over QuickSort and MergeSort, two popular sorting algorithms used in modern applications

QuickSort

The Quicksort algorithm is an O(NLog(N)), in place, and not stable sorting algorithm with its default implementation. QuickSort partitions an array or list into 2 sections and correctly places a pivot in the sorted array. The partition is a O(n) algorithm which is recursively called on the left partition and the right partition. This equates to log n times calls on average in QuickSort's default implementation.


MergeSort

The MergeSort algorithm is an O(NLog(N)), not in place, stable sorting algorithm with its default implementation. MergeSort recursively sorts a list by dividing the array up into smaller and smaller chunks and using a O(N) merge algorithm to join them back together in sorted order. This requires O(N) extra space its MergeSort's default implementation which means the algorithm is not in place.
